# AI Sentiment Automation Pipeline
**Built by Vidhya Chenji**

### 🚀 Overview
This project automates the analysis of customer feedback using **Google Gemini 1.5 Flash**. It takes raw text data, determines the customer's sentiment, and generates a structured report for business decision-making.

### 🛠️ Tech Stack
* **Language:** Python
* **AI Model:** Vertex AI (Gemini 1.5 Flash)
* **Infrastructure:** Google Cloud Platform (GCP)
* **Version Control:** Git & GitHub

### 📈 Business Impact
Instead of manual review, this tool can process thousands of feedback entries in seconds, identifying frustrated customers immediately so the team can take action.
