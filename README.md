# **📄 Gemini PDF Intelligence & Summary Bot**
### **Built by Vidhya Chenji**

🚀 Overview
This project leverages Google Gemini 1.5 Pro/Flash to process complex PDF documents. It automates the extraction of key insights, allowing users to summarize long papers or ask specific questions about document contents in seconds.

🛠️ Tech Stack
Language: Python

AI Model: Vertex AI (Gemini 1.5 Pro)

Infrastructure: Google Cloud Platform (GCP)

Version Control: Git & GitHub
